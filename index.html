Кнопки

Общий класс для кнопок btn

Можно использовать с тегами a, div, button

Размеры кнопок

Предусмотрено три размера кнопок:

  • Микро btn--micro
  • Маленький btn--s
  • Стандартный btn
  • Большой btn--l

Виды кнопок

Обычная кнопка btn

Прозрачная кнопка btn btn--ghost

Кнопка с svg иконкой btn

Прозрачная кнопка с svg иконкой btn btn--ghost

Кнопка с тенью btn btn--shadow

Кнопка для интрефейсов btn btn--ui, используются без текстовых подписей

Группировка кнопок

для выстраивания кнопок в линию используется класс для блочного div btn-pan

Кнопки в ряд с правым отступом между ними, высота кнопок подстраивается под наибольшую btn-pan
Ряд кнопок с выравниванием по базовой линии, без подстройки высоты btn-pan btn-pan--baseline
Кнопки в ряд с выравниванием по центру btn-pan bnt-pan--center
Кнопки в ряд с выравниванием по правому краю btn-pan bnt-pan--right
Кнопки в колонке btn-pan bnt-pan--col
панель c интерфейсными кнопками btn-pan > btn-ui
панель c интерфейсными кнопками btn-pan > btn-ui btn--s
